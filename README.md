# 🤖 Kuro — Multifunctional Discord Bot

Kuro is a feature-rich Discord bot built to handle moderation, utility tasks, and fun interactions — all in one place.

---

## 🚀 Features

### 🛡️ Moderation

* Censor messages
* Timeout / Remove Timeout
* Ban / Unban users
* Kick members

### 🎭 Role Management

* Assign roles
* Remove roles

### 🧰 Utilities

* Fetch user avatars
* Convert JPG/PNG to GIF
* Set nicknames
* Member count
* Lock / Unlock channels
* Slowmode control

### 🧮 Tools

* Calculator
* Poll system

### 🎉 Fun

* Flip a coin
* Custom Tic Tac Toe game (interactive buttons)

### 🤖 AI Integration

* AI-powered replies
* Web search functionality

### 🔔 Automation

* Welcome messages when members join
* Ban list tracking

---

## 🕹️ Games

### Tic Tac Toe

* Interactive button-based UI
* Turn-based gameplay
* Win & draw detection
* Multiplayer support

## ⚙️ Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/kuro-bot.git
cd kuro-bot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Create `.env` file

```env
DISCORD_TOKEN=your_bot_token_here
```

### 4. Run the bot

```bash
python main.py
```
